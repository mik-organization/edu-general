# データベースを使用するWebアプリケーションの構成図
```plantuml
@startuml
!define RECTANGLE class

RECTANGLE Server {
  + IP: String
  + OS: String
}

RECTANGLE Client {
  + IP: String
  + OS: String
}

RECTANGLE Database {
  + DBName: String
  + DBType: String
}

Client --> Server : HTTP Request
Server --> Database : SQL Query

@enduml
```

# Webアプリケーションの配置図
```plantuml
@startuml
node "Client" {
  [Browser]
}

node "Server" {
  [Web Server]
  [Application Server]
}

node "Database" {
  [DB Server]
}

[Browser] --> [Web Server] : HTTP Request
[Web Server] --> [Application Server] : Internal Call
[Application Server] --> [DB Server] : SQL Query

@enduml
